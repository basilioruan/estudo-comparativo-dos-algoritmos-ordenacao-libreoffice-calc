# estudo-comparativo-dos-elgoritmos-de-ordena-o-do-libreoffice-calc
Estudo empírico e código-fonte em C++ para comparar o desempenho de algoritmos de ordenação (Insertion Sort, Shell sort, Quick Sort, Introsort, std::sort). O objetivo é validar o uso do algoritmo Introsort no LibreOffice Calc, com foco na eficiência e escalabilidade em grandes volumes de dados.
